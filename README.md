# VSCode Monokai for Zed

A Monokai color theme for [Zed](https://zed.dev/), based on the classic VSCode Monokai theme.

## Themes

This extension includes two theme variants:

- **Monokai** - The classic Monokai theme with vibrant colors
- **Monokai Dimmed** - A softer, more muted version of Monokai

## Installation

1. Open Zed
2. Open the command palette (`Cmd+Shift+P` on macOS, `Ctrl+Shift+P` on Linux)
3. Enter `zed: extensions`
4. Search for "Monokai" and install
5. Open the theme selector (`theme selector: toggle`) and choose your preferred Monokai variant

## Color Palette

### Monokai

| Color | Hex | Usage |
|-------|-----|-------|
| Background | `#272822` | Editor background |
| Foreground | `#f8f8f2` | Default text |
| Yellow | `#E6DB74` | Strings |
| Orange | `#FD971F` | Function parameters |
| Pink | `#F92672` | Keywords |
| Purple | `#AE81FF` | Numbers, constants |
| Green | `#A6E22E` | Functions, class names |
| Cyan | `#66D9EF` | Types, library functions |
| Comment | `#88846f` | Comments |

### Monokai Dimmed

| Color | Hex | Usage |
|-------|-----|-------|
| Background | `#1e1e1e` | Editor background |
| Foreground | `#c5c8c6` | Default text |
| Green | `#9AA83A` | Strings |
| Blue | `#6089B4` | Variables, numbers |
| Orange | `#CE6700` | Function names |
| Purple | `#9872A2` | Storage, keywords |
| Red | `#C7444A` | Support, classes |
| Gray | `#676867` | Keywords |
| Comment | `#9A9B99` | Comments |
